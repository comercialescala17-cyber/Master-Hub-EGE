▸ Setup — 6 passos
01
Instalar n8n
npx n8n ou docker run -it --rm -p 5678:5678 n8nio/n8n
02
Importar workflows
Settings → Import workflow → selecione ege-message-flow.json e ege-cron-kpi.json
03
Criar credenciais
Credentials → New: "Moskit API Key" (httpHeader), "Slack EGE" (slackApi)
04
Configurar envs
Settings → Environment Variables → adicione todas as variáveis marcadas como obrigatórias
05
Configurar webhook
Ative o workflow, copie a URL do Webhook e configure como destino na Evolution API
06
Ativar workflows
Toggle "Active" em ambos os workflows — KPI cron dispara automaticamente às 08h
